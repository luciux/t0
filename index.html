Componentes Curriculares e Trabalhos

1 Semestre

  • Matemática Instrumental;
  • Introdução à Informática;
  • Circuitos Digitais;
  • Leitura e Produção textual;
  • Algoritmos e Programação;

O trabalho principal desde semestre foi integrado entre os componentes curriculares de Matemática Instrumental e Algoritmos e Programação, cuja o mesmo tinha por objetivo modelar uma função matemática através de um algoritmo, gerar as saídas e mostrar um gráfico. Sendo assim treinando as habilidades de programação e revendo os conceitos matemáticos, utilizando-os para montar o algoritmo e analisar as saídas.

2 Semestre

  • Estatística básica
  • Leitura e produção textual II
  • Estrutura de dados I
  • Sistemas digitais
  • Cálculo I

Os trabalhos desenvolvidos neste semestre foram diversos sendo os principais o do componente curricular de estrutura de dados, desenvolvendo listas, simples e duplas, com o objetivo de utilizar métodos de ordenação para organiza-las, também como aprimorar a utilização estruturas dentro da linguagem de programação C.

3 Semestre

  • Matemática discreta
  • Estrutura de dados II
  • Programação I
  • Probabilidade e estatística
  • Cálculo II
  • Organização de computadores

No terceiro semestre foi adquirido conhecimentos em novas linguagens de programação, e novas técnicas de programação, como por exemplo a programação Orientada a Objetos, a qual foi posta em prática com a linguagem Java. Em Organização de Computadores utilizamos uma linguagem de mais baixo nível, o Assembly. Ainda foi desenvolvida uma aplicação para o S.O Android.

4 Semestre

  • Banco de dados I
  • Programação II
  • Cálculo numérico
  • Introdução ao pensamento social
  • Teoria da computação
  • Grafos

Interesses

O meu objetivo enquanto acadêmico da instituição é aproveitar o máximo dos componentes curriculares, participar em congressos e feiras com intuito de conhecer novas tecnologias e vivenciar experiências diferentes. A expectativa para a matéria de programação II é se tornar um bom programador web, e realizar um trabalho integrado com banco de dados que tenha uma funcionalidade além de servir como método avaliativo do componente.
